3-Tier Responsive Design: 720px and 600px breakpoints

Desktop Layout
> 720px (750px shown)
Full Desktop: 2 panes top (Console → Store) | 2 panes bottom (Query selector | Terminal output)
Console commands display cleanly, terminal has adequate width for output tables.
Hybrid Layout
720px - 600px (710px shown)
Tablet/Small Desktop: 2 panes top (Console → Store) | 1 pane + 1 pane bottom (Query selector stacked above Terminal)
Console stays horizontal, but terminal gets full width for better readability.
Mobile Layout
< 600px (590px shown)
Mobile: 4x 1 panes - everything stacked vertically
Console text wraps properly, arrow hidden, optimized for narrow screens.
